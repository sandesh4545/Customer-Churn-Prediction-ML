# End-to-End Customer Churn Prediction System

## Overview
This project predicts customer churn using machine learning models.

## Dataset
Telco Customer Churn Dataset (Kaggle)

## Models Used
- Logistic Regression
- Random Forest

## Results
Logistic Regression achieved:
- Accuracy: 81.6%
- ROC-AUC: 0.739

## Conclusion
The model helps telecom companies identify customers at risk of churn and take preventive retention actions.
